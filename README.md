<a> This is a repo that stores our source code for the furniture selling program for a school group project. <br>
The school project has since been finished and now it sits abandoned (but unarchived) until further use. </a>

<a> If you wanted this program to run, please clone the repository and open it as a NetBeans project (or an IDE of your choice) and run it there. </a>

<a> This program requires JDK 21 and above (or change that in the NetBeans properties) </a>
